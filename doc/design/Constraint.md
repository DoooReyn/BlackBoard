# 编程规范

## 前缀

- 类型声明使用 `T` 前缀，比如：`TEngineStates`
- 接口声明使用 `I` 前缀，比如：`IEngineOptions`
- 枚举声明使用 `E` 前缀，比如：`ESystemEvent`
- 特别地，如果前缀与首字母相同，那么可以不使用前缀，比如：`I18n`

## 命名

- 变量
    - 局部变量使用蛇形命名法，单词之间使用 `_` 连接
    - 类私有变量使用小驼峰，且开头添加 `_`
    - 其他情况一律使用小驼峰
- 类
    - 类的名称使用大驼峰
    - 类的公开方法名称使用小驼峰
    - 类的保护/私有方法名称使用小驼峰，且开头添加 `_`
- 其他方法
    - 一律使用小驼峰
